创客与编程-机械臂指令说明

最多支持6个参数,角度值和坐标值支持小数点后 1位,如下
命令1;命令2 参数1,参数2;命令3 参数1,参数2,参数3, , ,参数6
返回 "Cmd" 非0 代表命令被放进 指令池 排队等待执行,当前得到的各舵机角度值并非最终结果

// 注释符号后面的整行内容,不执行
? {"X":"90.0","Y":"90.0","Z":"90.0","B":"90.0","T":"90.0","E":"90.0","Cmd":0}
? H.txt 查看指定文件内容,获取设备名称 ? name; ? IP; ? F;

format 格式化命令,删除控制板所有文件,要重新 [上传文件]
dir 列出所有文件
re 重启开发板
stop 0 立即停止动作,并清除所有剩余未执行指令
stop -1 立即暂停动作,需要恢复用 Start; 
stop 5000 强行暂停5秒后自动恢复动作
Start 立即恢复之前被暂停时未执行完的动作
sync  1 默认 1 是同步,异步为 sync 0;

F 指定当前文件为默认的 Auto.txt
F a.txt 指定当前为 a.txt 之后 C 和 S 和 D 都对这个文件操作
C a.txt 删除  a.txt  文件,不指定文件名则 删除 F 指定文件
S 往 F 指定文件末尾追加一条所有舵机角度状态 XYZ指令
S 5000 往 F 指定文件末尾追加一条延时5秒指令 delay 5000
S X 90 往 F 指定文件末尾追加一条动作指令  X 90
D 3 删除 F 指定文件末尾三行指令,D 删除一行
D -1 删除 F 指定文件里的所有内容
SH 保存当前所有舵机角度状态到 H.txt
step 0 原速执行 X  Y  Z  B  T  E 指令,  XYZ 指令永远是减速的
step 1 减速执行 X  Y  Z  B  T  E 指令
delay 5000 延时5秒后继续往下执行

H 执行 H.txt 动作脚本文件
X 50.5 让X转到50绝对角度, 范围 0度-180度
Y ? 让Y转到 ? 随机角度.  可活动角度受限于机械结构配置
Z -10 让Z以当前角度转到-10度的相对角度,范围受限于配置
E +5
 让E以当前角度转到 +5度的相对角度,范围受限于配置
XYZ +10,?,,,,-5 以各舵机当前角度相对移动,X+10度,Y随机,ZBT不变,E-5度
XYZ 90,90,90,90,90,80 同时转动XYZBTE到指定角度,多舵机同时转动耗电大要外部供电

time 列出 倒计时 和 定时 设定,共用最多10个
time -1 删除所有 倒计时 和 定时 设定
time 5 删除 5号 倒计时 或 定时 设定,范围 0-9
time 5000,H.txt,1 设定倒计时5秒后执行H.txt一次
time 2024-10-1 12:1:0 H.txt 设置定时执行 H.txt 一次.要连路由器WIFI到外网才有效
time ?-?-? 12:?:0 H.txt 60 不论年月日,12点时每分钟都执行 H.txt 一次,总共60次

HTTP接口+改成_ GET 或 POST http://IP/cmd?x 80&y _10&E -10&time=时间随机值
电脑COM控制需要安装 CH340驱动,选择COM端口,选115200波特率,发文本型命令控制
支持 TCP客户端与远程服务器 UDP协议 blinker.点灯物连网 控制,端口配置进 [系统设置]
