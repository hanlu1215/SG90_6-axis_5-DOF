?			{"X":"90.0","Y":"50.0","Z":"180.0","E":"90.0","Cmd":0}
? name		设备名称AP_SSID
? H.txt		查看指定文件内容
? IP
? F

dir			列出所有文件
format		删除所有文件,要重新 [上传文件]
re			重启开发板
stop 0		立即停止动作,并清除所有剩余未执行指令
stop -1		立即暂停动作,需要用 Start 恢复
stop 5000	强行暂停5秒后自动恢复动作
Start		立即恢复之前被暂停时未执行完的动作
sync  1           默认是同步,异步为sync 0


F			指定当前文件为默认的 Auto.txt
F a.txt		指定当前文件为 a.txt 之后的 C 和 S 和 D指令都对这个文件操作
C			删除 F 指定文件
C a.txt		删除  a.txt  文件
S			往 F 指定文件末尾追加一条所有舵机角度状态的 XYZ 指令
S 5000		往 F 指定文件末尾追加一条延时5秒指令 delay 5000
S X 90		往 F 指定文件末尾追加一条动作指令  X 90
D 3			删除 F 指定文件末尾三行指令,D 删除一行
D -1			删除 F 指定文件里的所有内容
SH			保存当前所有舵机角度状态到 H.txt

step 0		原速执行 X  Y  Z  E 指令,  XYZ 和 GO永远是减速的
step 1		减速执行 X  Y  Z  E 指令
delay 5000	延时5秒后继续往下执行

H			执行 H.txt 动作
X 50.5		让X转到50绝对角度, 范围 0度-180度
Y ?			让Y转到 ? 随机角度.  可活动角度受限于机械结构配置
Z -10		让Z以当前角度转到-10度的相对角度,范围受限于配置
E +5		让E以当前角度转到 +5度的相对角度,范围受限于配置
XYZ +10,?, , , ,-5		以各舵机当前角度进行相对移动,X+10度,Y随机,ZBT不变,E-5度
XYZ 90,90,90,90,90,80		同时转动XYZBTE到指定角度,多舵机同时转动耗电大要外部供电

time						列出 倒计时 和 定时 设定,共用最多10个
time -1						删除所有 倒计时 和 定时 设定
time 5						删除 5号 倒计时 或 定时 设定,范围 0-9
time 5000,H.txt,1				设定倒计时5秒后执行H.txt一次
time 2024-10-1 12:1:0 H.txt	设置定时执行H.txt共十次
time ?-?-? 12:?:0 H.txt 60		设置不管年月日,12点钟时每分钟都执行 H.txt 一次,总共60次
